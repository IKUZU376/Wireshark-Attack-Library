# ICMP Traffic Analysis

## Objective

Capture and analyze ICMP traffic generated by the `ping` utility.

---

## Tools Used

* Wireshark
* Ping Utility

---

## Command Used

```bash
ping -c 10 8.8.8.8
```

---

## Observations

* ICMP Echo Requests were sent to the target host.
* ICMP Echo Replies were received successfully.
* Packet exchange confirmed network connectivity.

---

## Protocols Involved

* ICMP
* IPv4

---

## Security Relevance

ICMP traffic is commonly used for:

* Network troubleshooting
* Host discovery
* Connectivity testing
* Reconnaissance activities
